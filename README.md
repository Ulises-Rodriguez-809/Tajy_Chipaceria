# Tajy Chipaceria

Pagina informativa sobre la chipaceria Taji, en la cual se podra ver toda la informacion sobre este local de comida.
Pagina totalmente responsive adaptada a todas las resoluciones.

**Intalacion:**

1. Clone el repositorio de GitHub: git clone https://github.com/Ulises-Rodriguez-809/Tajy_Chipaceria
2. Cambie al directorio del proyecto: cd mi-proyecto-de-ejemplo
3. Instale las dependencias: npm install

**Uso:**

1. En el terminal: npm run dev
2. Abra en su navegador: http://localhost:5173/

**Metodologia de trabajo:**

1. Llmadas constantes para manterner una clara comunicacion y evitar confuciones
2. SCRUM para la divicion de trabajos

**Tecnologias:**

1. React JS como libreria base
2. React Vite para la estructura base de la aplicacion
3. Bootstrap para el estilo de componentes y manejo de vistas responsive
